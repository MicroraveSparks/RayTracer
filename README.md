# RayTracer

This is a project inspired by the [RayHunter](https://efforg.github.io/rayhunter/) project by the [Electronic Frontiers Foundation](https://www.eff.org/), made for a CS453 Honors project. It is an IMSI-catcher detector that utilizes:
- 1x NodeMCU ESP32 Development board
- 1x Quectel EG25-G USB Module
- 1x HW-125 Micro SD Card Module
    - Requires SD card with $\geq$ 1MB of storage
